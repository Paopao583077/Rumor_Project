# 代码原理

### 第一步：数据准备 (`dataset.py`)

**原理回顾：** 模型看不懂汉字和图片，必须把它们变成“数字矩阵”（Tensor）。

#### 核心代码片段解析：

```
# 对应 dataset.py 中的 __getitem__ 函数

def __getitem__(self, index):
    # 1. 获取原始文本和图片路径
    text = self.df.iloc[index]['text']
    image_path = self.df.iloc[index]['image_path']

    # ---------------------------------------------------------
    # 原理对应：文本 Tokenization (分词与编码)
    # 数学含义：将句子 S 转化为 ID 序列 [101, 321, ...]
    # ---------------------------------------------------------
    encoding = self.tokenizer.encode_plus(
        text,
        add_special_tokens=True,    # 加上 [CLS] 和 [SEP]
        max_length=128,             # 固定长度，不够补0，超了截断
        padding='max_length',
        truncation=True,
        return_tensors='pt'         # 返回 PyTorch 张量
    )
    
    # input_ids: 每个字的身份证号
    # attention_mask: 告诉模型哪些是真字(1)，哪些是补零的废话(0)
    input_ids = encoding['input_ids'].squeeze()       
    attention_mask = encoding['attention_mask'].squeeze()

    # ---------------------------------------------------------
    # 原理对应：图像预处理 (Normalization)
    # 数学含义：I = (I_raw - Mean) / Std
    # ---------------------------------------------------------
    image = Image.open(image_path).convert('RGB') # 强转3通道，解决PNG透明问题
    image = self.transform(image)                 # 缩放至 224x224 并归一化

    return {
        'input_ids': input_ids,
        'attention_mask': attention_mask,
        'image': image,
        'label': torch.tensor(label, dtype=torch.long)
    }
```

### 第二步：模型架构 (`model.py`) —— **最核心部分**

**原理回顾：** 双塔结构 + 注意力融合。

#### 1. 初始化部分 (`__init__`)

这里定义了模型的“零部件”。

Python

```
class RumorDetector(nn.Module):
    def __init__(self):
        super(RumorDetector, self).__init__()
        
        # 零部件 A: 文本编码器 (BERT)
        # 对应原理：h_text = BERT(T)
        self.bert = BertModel.from_pretrained('bert-base-chinese')
        
        # 零部件 B: 图像编码器 (ResNet)
        # 对应原理：h_img = ResNet(I)
        resnet = models.resnet50(pretrained=True)
        # 去掉 ResNet 最后分类的那一层，我们只要它的特征提取部分
        self.resnet = nn.Sequential(*list(resnet.children())[:-1]) 
        
        # 零部件 C: 融合层 (全连接层做维度对齐)
        # 对应原理：把 BERT(768维) 和 ResNet(2048维) 统一映射到 256维
        self.text_fc = nn.Linear(768, 256)   # 生成 Query
        self.image_fc = nn.Linear(2048, 256) # 生成 Key 和 Value
        
        # 零部件 D: 分类器
        # 对应原理：最终判决 y = Wx + b
        self.classifier = nn.Linear(768 + 256, 2) # 输入是拼接后的维度，输出是2(谣言/非谣言)
```

#### 2. 前向传播部分 (`forward`)

这里定义了数据怎么流转，也是**数学公式发生的地方**。

Python

```
    def forward(self, input_ids, attention_mask, images):
        # ==================== 1. 特征提取 ====================
        
        # 文本塔：提取 BERT 的 [CLS] 向量
        # 对应公式：h_text = BERT(T)
        bert_output = self.bert(input_ids=input_ids, attention_mask=attention_mask)
        text_features = bert_output.pooler_output  # 维度: [Batch, 768]
        
        # 视觉塔：提取 ResNet 特征
        # 对应公式：h_img = ResNet(I)
        image_features = self.resnet(images)
        image_features = image_features.view(image_features.size(0), -1) # 展平: [Batch, 2048]
        
        # ==================== 2. 注意力融合 (Text-Guided Attention) ====================
        
        # 第一步：线性变换 (Projection)
        # 对应公式：Q = h_text * Wq
        query = self.text_fc(text_features).unsqueeze(1)  # [Batch, 1, 256]
        # 对应公式：K = h_img * Wk, V = h_img * Wv
        key = self.image_fc(image_features).unsqueeze(1)  # [Batch, 1, 256]
        value = self.image_fc(image_features).unsqueeze(1) # [Batch, 1, 256]
        
        # 第二步：计算分数 (Dot Product Attention)
        # 对应公式：Score = Q * K^T
        # torch.bmm 是批量矩阵乘法
        attention_scores = torch.bmm(query, key.transpose(1, 2)) 
        
        # 第三步：归一化 (Softmax)
        # 对应公式：alpha = Softmax(Score / sqrt(d))
        attention_weights = F.softmax(attention_scores / (256**0.5), dim=-1)
        
        # 第四步：加权求和
        # 对应公式：v_att = alpha * V
        fused_visual = torch.bmm(attention_weights, value).squeeze(1) # [Batch, 256]
        
        # ==================== 3. 分类决策 ====================
        
        # 特征拼接 (Concatenate)
        # 对应公式：v_final = Concat(h_text, v_att)
        combined_features = torch.cat((text_features, fused_visual), dim=1) # [Batch, 768+256]
        
        # 全连接分类
        # 对应公式：y = Softmax(W * v_final + b)
        logits = self.classifier(combined_features) # [Batch, 2]
        
        return logits
```

### 第三步：训练循环 (`train.py`)

**原理回顾：** 计算 Loss，反向传播，更新参数。

```
# 对应 train.py 中的训练循环

# 1. 定义损失函数 (交叉熵 Cross Entropy)
# 对应原理：L = -sum(y * log(y_pred))
criterion = nn.CrossEntropyLoss()

# 2. 定义优化器 (AdamW)
# 对应原理：参数更新规则 theta_new = theta_old - lr * gradient
optimizer = AdamW(model.parameters(), lr=2e-5)

for epoch in range(EPOCHS):
    for batch in train_loader:
        # A. 清空之前的梯度 (防止累加)
        optimizer.zero_grad()
        
        # B. 前向传播 (Forward Pass)
        # 这行代码会跑一遍上面 model.py 里的 forward 函数
        logits = model(input_ids, attention_mask, images)
        
        # C. 计算损失 (Calculate Loss)
        # 比较 模型的预测(logits) 和 真实答案(labels) 的差距
        loss = criterion(logits, labels)
        
        # D. 反向传播 (Backpropagation)
        # 对应原理：链式法则求导。
        # PyTorch 会自动计算 Loss 对每一个参数(W, b)的导数
        loss.backward()
        
        # E. 更新参数 (Optimizer Step)
        # 对应原理：根据导数修改模型权重，让它下次猜得更准
        optimizer.step()
```

### 第四步：预测与评估 (`gui_app.py` / `predict.py`)

**原理回顾：** 此时模型已经学会了（权重固定了），只做前向推理。

```
# 对应 gui_app.py 中的 predict 函数

# 1. 开启评估模式
# 重要！这会关闭 Dropout 层，保证预测结果稳定
model.eval() 

# 2. 关闭梯度计算
# 告诉 PyTorch：这次只是做题，不用复习(不更新参数)，省点内存
with torch.no_grad():
    # 3. 直接获取分数
    logits = model(input_ids, attention_mask, image_tensor)
    
    # 4. 转化为概率 (Softmax)
    # 例如 logits 是 [2.5, -0.5] -> 经过 softmax -> [0.95, 0.05]
    probs = torch.softmax(logits, dim=1)
    
    # 5. 拿出最大值的索引 (0 或 1)
    pred = torch.argmax(probs, dim=1).item()
```
