## 第二步：数据清洗 (`preprocess.py`)

**目的：** 将原始乱糟糟的 `.txt` 文件转换为机器易读的 `.csv` 表格。

** 原理图解：**

Code snippet

```
graph LR
A[原始TXT文件] -->|读取每3行| B(提取图片文件名)
A -->|读取每3行| C(提取微博文本)
B & C --> D{数据清洗}
D -->|匹配成功| E[生成 train.csv / test.csv]
```

**代码实现：**

Python

```
import os
import pandas as pd

def parse_raw_txt(rumor_file, nonrumor_file, output_csv):
    """
    功能：读取乱糟糟的txt，整理成整齐的csv
    """
    print(f"正在生成 {output_csv} ...")
    data_rows = []

    def process_file(filepath, label_value):
        if not os.path.exists(filepath):
            print(f"⚠️ 警告：找不到文件 {filepath}")
            return

        with open(filepath, 'r', encoding='utf-8') as f:
            lines = f.readlines()
        
        # 循环读取：步长为3，因为每3行是一条数据
        for i in range(0, len(lines), 3):
            if i + 2 >= len(lines): break
            
            # --- 1. 提取图片 ---
            image_line = lines[i+1].strip()
            image_urls = image_line.split('|')
            target_image_name = None
            for url in image_urls:
                if url != 'null' and url.endswith('.jpg'):
                    target_image_name = url.split('/')[-1] # 拿到 xxx.jpg
                    break
            
            if target_image_name is None: continue 

            # 图片路径统一指向 ./data/images/
            image_path = f"./data/images/{target_image_name}"

            # --- 2. 提取文本 ---
            text_content = lines[i+2].strip()
            if not text_content: text_content = "无文本"

            data_rows.append({
                'path': image_path,
                'text': text_content,
                'label': label_value
            })

    # 1 代表谣言，0 代表非谣言
    process_file(rumor_file, 1)      
    process_file(nonrumor_file, 0)   

    # 保存 CSV
    if len(data_rows) > 0:
        df = pd.DataFrame(data_rows)
        df = df.sample(frac=1, random_state=42).reset_index(drop=True) # 打乱顺序
        df.to_csv(output_csv, index=False)
        print(f"✅ 成功！已生成 {output_csv} (共 {len(df)} 条数据)")
    else:
        print("❌ 失败：没有提取到任何数据。")

if __name__ == '__main__':
    # 生成训练集
    parse_raw_txt('./data/tweets/train_rumor.txt', './data/tweets/train_nonrumor.txt', './data/train.csv')
    # 生成测试集
    parse_raw_txt('./data/tweets/test_rumor.txt', './data/tweets/test_nonrumor.txt', './data/test.csv')
```

** 操作：** 右键运行此文件，直到 `data` 目录下出现 `.csv` 文件。

## 第三步：制作“喂食器” (`dataset.py`)

**目的：** 定义数据如何被“喂”给神经网络。深度学习模型只吃数字，不吃图片文件。

**原理图解：**

Plaintext

```
原始数据          处理流程 (Pipeline)                  模型输入 (Tensor)
[图片.jpg]  --->  Resize(224x224) + Normalize  --->  [3, 224, 224] 矩阵
"震惊!..."  --->  BERT Tokenizer (查字典)      --->  [101, 234, 567...] 数字序列
```

**代码实现：**

Python

```
import torch
from torch.utils.data import Dataset
from PIL import Image
from transformers import BertTokenizer
import torchvision.transforms as transforms
import pandas as pd

class WeiboDataset(Dataset):
    def __init__(self, csv_file, max_len=128, mode='train'):
        self.data = pd.read_csv(csv_file)
        self.tokenizer = BertTokenizer.from_pretrained('bert-base-chinese')
        self.max_len = max_len
        
        # 图片预处理流水线
        self.transform = transforms.Compose([
            transforms.Resize((224, 224)), 
            transforms.ToTensor(),         
            transforms.Normalize(mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])
        ])

    def __len__(self):
        return len(self.data)

    def __getitem__(self, idx):
        row = self.data.iloc[idx]
        text = str(row['text'])
        img_path = row['path'] 
        label = int(row['label'])

        # 处理图片 (容错机制：坏图变黑图)
        try:
            image = Image.open(img_path).convert('RGB')
            image = self.transform(image)
        except:
            image = torch.zeros(3, 224, 224)

        # 处理文本
        encoding = self.tokenizer.encode_plus(
            text, add_special_tokens=True, max_length=self.max_len,
            padding='max_length', truncation=True, return_attention_mask=True, return_tensors='pt'
        )

        return {
            'input_ids': encoding['input_ids'].flatten(),
            'attention_mask': encoding['attention_mask'].flatten(),
            'image': image,
            'label': torch.tensor(label, dtype=torch.long)
        }
```

## 第四步：构建“大脑” (`model.py`)

**目的：** 定义神经网络的结构。这是系统的核心智能所在。

**架构图解：**

Plaintext

```
       [文本输入]                  [图片输入]
           ⬇                          ⬇
      (BERT 模型)                (ResNet 模型)
           ⬇                          ⬇
      [文本语义特征]              [视觉空间特征]
           ⬇                          ⬇
           ╰───────────┬──────────────╯
                       ⬇
            【 核心创新: 文本引导注意力模块 】
             (Text-Guided Attention)
           ➡ "让文本告诉眼睛该看图片的哪里"
                       ⬇
                  [融合后的特征]
                       ⬇
                  [分类器 (FC)]
                       ⬇
                 [输出: 谣言/非谣言]
```

**代码实现：**

Python

```
import torch
import torch.nn as nn
from transformers import BertModel
import torchvision.models as models

# --- 核心模块：注意力机制 ---
class TextGuidedAttention(nn.Module):
    def __init__(self, text_dim, img_dim, hidden_dim):
        super(TextGuidedAttention, self).__init__()
        self.text_proj = nn.Linear(text_dim, hidden_dim)
        self.img_proj = nn.Linear(img_dim, hidden_dim)
        self.v_proj = nn.Linear(img_dim, hidden_dim) 
        self.attention_score = nn.Linear(hidden_dim, 1)
        self.tanh = nn.Tanh()
        self.softmax = nn.Softmax(dim=1)

    def forward(self, text_feat, img_feat):
        t_proj = self.text_proj(text_feat).unsqueeze(1) 
        i_proj = self.img_proj(img_feat)
        combined = self.tanh(t_proj + i_proj) 
        scores = self.attention_score(combined).squeeze(-1)
        alpha = self.softmax(scores)
        v = self.v_proj(img_feat)
        return (alpha.unsqueeze(-1) * v).sum(dim=1)

# --- 完整模型 ---
class RumorDetector(nn.Module):
    def __init__(self, num_classes=2):
        super(RumorDetector, self).__init__()
        self.bert = BertModel.from_pretrained('bert-base-chinese')
        resnet = models.resnet50(pretrained=True)
        self.resnet_features = nn.Sequential(*list(resnet.children())[:-2])
        self.fusion_attention = TextGuidedAttention(768, 2048, 256)
        self.classifier = nn.Sequential(
            nn.Linear(768 + 256, 128), nn.ReLU(), nn.Linear(128, num_classes)
        )

    def forward(self, input_ids, attention_mask, image):
        text_feat = self.bert(input_ids=input_ids, attention_mask=attention_mask).pooler_output
        img_feat = self.resnet_features(image)
        batch, c, h, w = img_feat.size()
        img_feat_seq = img_feat.view(batch, c, -1).permute(0, 2, 1)
        
        attended_img = self.fusion_attention(text_feat, img_feat_seq)
        final_feat = torch.cat((text_feat, attended_img), dim=1)
        return self.classifier(final_feat)
```

## 第五步：开始训练 (`train.py`)

**目的：** 启动训练循环。让模型看着数据不断修正错误（反向传播），最终学会判断。

**代码实现 (适配 RTX 4060)：**

Python

```
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from dataset import WeiboDataset
from model import RumorDetector

def main():
    # 1. 自动检测显卡
    DEVICE = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    print(f"🚀 正在使用计算设备: {DEVICE}")
    
    # 2. 参数设置 (4060显卡建议设为 16 或 32)
    BATCH_SIZE = 16  
    LR = 2e-5       
    EPOCHS = 3      

    # 3. 加载数据
    print("📦 正在加载数据...")
    train_ds = WeiboDataset('./data/train.csv', mode='train')
    val_ds = WeiboDataset('./data/test.csv', mode='val')
    train_loader = DataLoader(train_ds, batch_size=BATCH_SIZE, shuffle=True)
    
    # 4. 初始化模型
    print("🧠 正在初始化模型...")
    model = RumorDetector().to(DEVICE)
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.AdamW(model.parameters(), lr=LR)
    
    # 5. 训练循环
    print("▶️ 开始训练！")
    for epoch in range(EPOCHS):
        model.train()
        total_loss = 0
        for i, batch in enumerate(train_loader):
            input_ids = batch['input_ids'].to(DEVICE)
            mask = batch['attention_mask'].to(DEVICE)
            images = batch['image'].to(DEVICE)
            labels = batch['label'].to(DEVICE)
            
            optimizer.zero_grad()
            logits = model(input_ids, mask, images)
            loss = criterion(logits, labels)
            loss.backward()
            optimizer.step()
            
            total_loss += loss.item()
            if i % 10 == 0:
                print(f"Epoch {epoch+1} | Step {i}/{len(train_loader)} | Loss: {loss.item():.4f}")

    # 6. 保存模型
    torch.save(model.state_dict(), 'best_model.pth')
    print("💾 训练完成，模型已保存为 best_model.pth")

if __name__ == '__main__':
    main()
```

**🚀 操作：** 右键运行 `train.py`，等待 5-15 分钟，直到出现 `best_model.pth`。
